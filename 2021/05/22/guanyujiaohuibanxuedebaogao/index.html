关于教会办学的报告

相关经文

以色列阿，你要听。耶和华我们神是独一的主。你要尽心、尽性、尽力爱耶和华你的 神。我今日所吩咐你的话都要记在心上，也要殷勤教训你的儿女。无论你坐在家里，行在 路上，躺下，起来，都要谈论。 也要系在手上为记号，戴在额上为经文。又要写在你房屋 的门框上，并你的城门上。(申 6:4-9)

教养孩童，使他走当行的道，就是到老他也不偏离。(箴 22:6 )

你们作父亲的，不要惹儿女的气，只要照着主的教训和警戒，养育他们。(弗 6:4 )

前言

守望教会坚持山上之城的异象到今日，教会公开化的努力已经进入尾声阶段。从现今外在环境来看，对于建堂，我们除了祷告，更多是在盼望中等待神开路，带领我们进入。在坚持异象之路的这十年间，从外在环境的变化和我们教会的需要这两个层面来看，教会下一代的教育日渐成为我们需要面对的问题。考虑办学，必然涉及与山上之城异象之间的关系。首先，办学也是山上之城异象的某种表达，与山上之城的异象并不冲突;其次，在现今盼望等候的过程中，办学还可能为下一个阶段教会的方向做一些探索和预备。

下面拟从三个方面简述教会办学的基本思路。

1、现状、需要与挑战

1. 教育的现状

在现今的环境下，父母对孩童学习环境的选择通常有四种:公立学校、私立学校、教
会学校、在家教育。让孩童进入到哪种学习环境中，最终的决定权在于父母。具体选择哪种方式，有多种因素需要考虑，父母需要根据家庭自身的能力、需要和信心来抉择。但无论选择哪一种方式，父母对孩童的责任都是第一位的，父母应当寻求上帝的心意，凭信心为孩童选择对他们最适合的方式，竭力将孩子培养成为敬虔的后裔。

2. 我们的需要

教会在教育中的角色是帮助父母，并与父母一同教养孩童。在以上提到的四种教育形式中，教会学校是教会可以更大参与和发挥作用的一种，却是目前我们缺少的一种。在一定条件下，教会可以办学。

这几年，全国各处有许多教会先后开始创办教会学校。在他们身上，我们看到上帝赐福的工作。虽然他们遭遇了很多艰难，但上帝的工作却不能被拦阻。相反，上帝借着他们让基督信仰更深地扎根于这块土地，影响其上的人们，向世界见证基督。

这一两年，整个世界局势发生了翻天覆地的变化。这一切都在提醒我们，我们生活在一个文化、信仰、价值观崩塌的混乱时代。面对时代的挑战，如何教养上帝所赐给我们的儿女，不仅成为许多父母的关注，也成了教会需要思考和面对的问题。我们需要思想在这种处境下，如何更好地培养孩童成为“尽心、尽性、尽力、尽意”爱主的人，能够应对时代的挑战和冲击，不但自己站立得稳，还能为上帝所用，成为周围人的祝福。

我们认为，在现今的环境中，教会应当更为主动地参与对孩童的教育和生命塑造，帮助父母竭力将孩童的人生建立在坚实的真理根基之上。因此，我们计划建立一个全日制的基督教学堂。

3. 现实的挑战

在现在的环境中，教会办学会面临以下几个方面的挑战。

首要的挑战是合一。孩童教育问题是一个敏感话题，不同的教育观念很可能会演化为互相纷争、彼此论断。我们当格外警醒，尽可能避免这种情况的发生。前面提到，一个家庭为孩童选择哪一种教育方式，涉及各种因素。盼望选择不同方式的家长和孩子，能更多互相尊重、彼此相顾，而不是互相轻视、彼此定罪。

此外，教会办学还有三个方面必须考虑的挑战。第一，家长信心的挑战。选择教会学堂的家长和孩子们要走一条与我们曾走过的，现在依然是社会主流的读书方式都不同的道路，这对于家庭的经济压力、父母的参与程度、孩童将来的工作就业等各方面都是很大的挑战。第二，外在压力的挑战。在对基督信仰打压力度日益加大的处境中，教会办学无疑是逆流而上，所面对的争战可想而知。因此，从教会到父母、孩童都需要做好面对各种可能打压的准备。第三，师资和教学本身的挑战。教育是一件专业的事情，不但需要经济上的持续投入，更需要专业精深的师资队伍。

要胜过这些挑战，我们惟有仰望上帝的恩典和帮助。

2、借鉴古典教育的思路

1.教育目标

教育的目的是塑造人。教会办学当以圣经为基础，按照主的教训，塑造尽心、尽性、尽意、尽力爱上帝，并且爱人如己的人(申 6:1-5;太 22:37-38;可 12:30)。

2.教育理念

为达成以上目标，拟借鉴基督教古典教育的理念，确立如下几点办学准则:

1) 以启示为根基

学堂的教学以认识上帝为基础，而认识上帝须通过上帝的启示(包括普遍启示和特殊 启示)，借着圣灵的开启。圣灵借着圣经让人认识基督，也借着万有和历史引导人认识基督。

2) 以基督为中心

基督是一切知识和智慧的目标，因祂是万有之主，拥有对一切领域的主权。所有知识有着彼此关联的内在统一性，并且基督是一切知识的中心。

3) 以传统为媒介

传统是人类智慧传承的方式，也是寻求智慧的重要途径。学堂注重三方面传统:教会
传统、西方文化传统和中国文化传统。

4) 以三艺为途径

三艺指语法、逻辑、修辞这三种学艺。语法不仅包括语言的形式和结构，也涵盖各门学科的基础知识和规则。逻辑训练人的思考能力，探索事物之间的相互关系，理解事物的原理。修辞则注重应用、实践和表达的智慧，将前两个阶段所学转化为输出，并创造性地应用在生活之中。

5) 以全人为目标

教育涉及人的各个方面，包括身体、情感、理性、意志等。学堂拟从身体、灵性、智性、品格等方面培育孩童，使他们成为健全的人。同时，学堂也考虑教授一些时代性的技能。

6) 以经典为材料

经典是文化传承最重要的载体，也是训练自由技艺最好的文本材料。但任何经典都要放在基督真理的光照之中来学习和评估。

7) 以汉语为优先

学堂拟采用双语学习，汉语作为母语当放在首要学习的位置，英文作为世界通用语言， 是重要的学习工具(古典语言适当的时候可以作为选修课)。

3、一些具体问题

1. 学堂和教会的关系

本着“教会办学、独立运营”的思路，学堂设立独立董事会，教会以拥有董事会固定席位等方式发挥影响力。

2. 教职人员

学堂启动的最低配置为校长、行政及几位全职教师。

3. 学生

学堂招生主要面对本教会，非本教会成员可酌情考虑。

4. 课程

主要考虑开设圣经、历史、语文、英语、数学和科学，及体育、艺术等科目。

5. 资金

资金来源主要来自教会拨款、学费和筹款。
