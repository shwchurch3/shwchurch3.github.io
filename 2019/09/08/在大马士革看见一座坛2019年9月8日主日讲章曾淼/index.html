在大马士革看见一座坛——2019年9月8日主日讲章/曾淼 传道

Podcast: Play in new window | Download


亲爱的弟兄姐妹：主日平安！

今天要来跟大家分享的这位犹大君王，可以说是一位“富三代”。他的爷爷乌西雅王在位期间富国强兵，他的父亲约坦王因遵行神的道，而日渐强盛(代下27:6)。这位少主二十岁登基，按说是有机会励精图治、更上一层楼的；然而在他作王的十六年中，他几乎把整个犹大国败光，并且为之后犹大的被掳奠定了不可逆转的基础。这位王就是亚哈斯，不仅名字跟北国以色列王亚哈只差一个字，作风也十分相像。亚哈斯“不像他祖大卫行耶和华他神眼中看为正的事。却效法以色列诸王所行的，又照着耶和华从以色列人面前赶出的外邦人行可憎的事，使他的儿子经火，并在邱坛上、山冈上、各青翠树下献祭烧香。”(王下16:2b-4) 亚哈斯的败国，首先始于他肆无忌惮的偶像崇拜。使儿子经火，是向摩洛神献祭过程中的做法。弥尔顿在《失乐园》中称摩洛是第一个加入撒旦大军的空中掌权者。他这样写道：第一个降临人间的可怕魔王摩洛 / 以人为牲，全身沾满了人血和亲人的眼泪 / 孩子在狰狞恐怖的恶魔手中的哭喊声，淹没在大鼓、小鼓的嚣声中 / 亚扪人崇拜他，并在拉巴及其沟渠纵横的原野 / 在亚珥歌伯、巴珊，直到亚嫩河滨，供奉着他 / 他不满足于这些淫邪的邻国 / 还使用奸计，迷惑了绝顶聪明的所罗门 / 在不洁的橄榄山上 / 面对真神的圣殿，立庙建祠 / 又把秀丽的欣嫩子谷封上，尊为圣林 / 因此被称为陀斐特和格痕拿 / 那就是阴间地狱的称号。亚哈斯身体力行地参与对迦南异教的偶像崇拜，使之重新成为犹大官方所认可的宗教活动。这就是为什么先知以赛亚一开篇就在天地面前责备犹大为“犯罪的国民，担着罪孽的百姓；行恶的种类，败坏的儿女”，因为那时候的犹大人“离弃耶和华，藐视以色列的圣者，与他生疏，往后退步。”(赛1:4) 因着他们离弃了耶和华他们列祖的神，耶和华就将他们交在亚兰王手里、交在以色列王手里(代下28:5-6)。而亚哈斯在面对他的困局之时选择继续堕落，犹大在他的手中也随之全面腐化。亚哈斯的一生让我们看到，在敬拜上离弃真神，就是把自己交在那恶者的手中，最终只会让人沦为私欲的奴隶。

一、四面受敌

第5节说，“亚兰王利汛和以色列王利玛利的儿子比加上来攻打耶路撒冷，围困亚哈斯，却不能胜他。” 亚哈斯后续的故事，就是以这场“亚兰-以法莲战争”作为背景。那时，亚兰王还收回了曾经被乌西雅王收回的出海口以拉他，并将犹大人从以拉他赶了出去(6)。以拉他几次易主，最后犹大还是没能保住。作者说：“亚兰人就来到以拉他，住在那里，直到今日”(6)，就是在向当时的读者解释以拉他是如何形成了今天这样的格局。我们听了这么多列王的事迹，已经摸到了这个规律，就是君王若遵守神的道，就争战得胜、扩张疆界、四境平安；反之，若拜偶像离弃神，就遭攻打、领土被占，甚至百姓被掳。然而即便是后者，那当中也仍然有神的怜悯，就是藉着先知、祭司的口，邀请君王悔改、转向神、寻求神的帮助。亚兰和以色列上来攻打耶路撒冷、围困亚哈斯，一方面是神出手管教亚哈斯；但是另一方面，他们却不能胜他，就显出神在给亚哈斯时间，也在给他机会作出正确的回应。从以赛亚书中我们得知，当听到“亚兰与以法莲已经同盟”的消息时，“王的心和百姓的心就都跳动，好像林中的树被风吹动一样”(赛7:2)，说明他们心里十分害怕。于是，耶和华派以赛亚去迎接亚哈斯，对他说：“你要谨慎安静，不要因亚兰王利汛和利玛利的儿子这两个冒烟的火把头所发的烈怒害怕，也不要心里胆怯……”(赛7:4)，然后晓谕亚哈斯说：“你向耶和华你的神求一个兆头，或求显在深处，或求显在高处。”(赛7:10-11) 意思就是说，耶和华邀请亚哈斯求一个超自然的神迹，好使他在面对这困境时，能相信神、且因信靠神而站立得住，最终能够化险为夷。弟兄姐妹，这就是神向我们所怀的赐平安的意念—他审判罪恶，但为的是要施行怜悯、挽回人心。对于神所发出的这个邀请，亚哈斯断然拒绝，而且还用了一个非常属灵、堂而皇之的理由：“我不求；我不试探耶和华。”(赛7:12) 意思是，不劳耶和华大驾了，其实他心里已经打定了主意，要抱亚述的大腿。这样一来，亚哈斯不吃敬酒就只能吃罚酒了—“因此，主自己要给你们一个兆头，必有童女怀孕生子，给他起名叫以马内利。到他晓得弃恶择善的时候，他必吃奶油与蜂蜜。因为在这孩子还不晓得弃恶择善之先，你所憎恶的那二王之地必致见弃。耶和华必使亚述王攻击你的日子临到你和你的百姓并你的父家。自从以法莲离开犹大以来，未曾有这样的日子”(赛7:14-17)。也就是说，用不了几年，亚兰和以色列就都要灭亡，而犹大也必将遭到亚述的攻击。然而亚哈斯眼瞎到，根本看不见、或者说无视背后究竟是谁在主导着列国的兴衰沉浮。

二、看见一座坛

回到列王纪的故事当中，亚哈斯差遣使者去见亚述王提革拉毗列色，说：“我是你的仆人、你的儿子。现在亚兰王和以色列王攻击我，求你来救我脱离他们的手。”(7) 在古代近东，“我是你的仆人、你的儿子”是君王加冕时向他们所敬拜的天上之神表示降服的说法。诗篇第二篇里面，受膏者说：“我要传圣旨。耶和华曾对我说：’你是我的儿子，我今日生你’。”(诗2:7) 亚哈斯用这样的说法，就表示他心甘乐意地向亚述王附庸称臣，甚至是以敬拜神的规格来尊崇。接着，他将耶和华殿里和王宫府库里的金银都送给亚述王为礼物。如果说亚哈是自卖，亚哈斯就是把自己卖给亚述。这样，亚述王答应了亚哈斯的请求，攻取了亚兰的大马士革，杀了亚兰王利汛，把居民掳到吉珥，其实这本来就是亚述在向西扩张的过程中所要做的。

接下来的一幕，是亚哈斯上大马士革去拜见他的新领主亚述王提革拉毗列色，一方面是庆贺军事行动的胜利，另一方面亚哈斯达到了自己的目的，为此去向亚述王表达感激之情。在这个过程中，亚哈斯很有可能跟着亚述王进到他们的大庙，在那里一起给亚述所拜的神献祭。就在那一刻，亚哈斯深深地被那座亚述人的坛所吸引了。哇，这座坛的样式是那么新颖、好看，使用起来是那么得体、方便，比我们的那座坛好多了！这么一比，我们的坛太落后了，必须得更新换代、换成这座！于是亚哈斯照着那座坛的样子画了图样，送到祭司乌利亚那里；祭司乌利亚立马领会王的精神，加班加点制作，赶在王还没回来之前，就把这座跟大马士革一模一样的新坛给建筑好了。“王从大马士革回来看见坛，就近前来，在坛上献祭，烧燔祭、素祭、浇奠祭，将平安祭牲的血洒在坛上，又将耶和华面前的铜坛，从耶和华殿和新坛的中间搬到新坛的北边。”(12-14)

亚哈斯的心其实是被什么吸引呢？是当他站在提革拉毗列色身边、与之分享得胜的荣耀和快感时，亚哈斯觉得这一切的权势、能力、荣华、甚至是幸福，都是由亚述人所敬拜的神带来的。这让他在看着那座坛的那一刻，心里生出无比的欣赏和羡慕。就好像一个运动员看到一双最新款的鞋，一个音乐人看到一件功能超强的乐器，一个男人看到一辆车，一个女人看到一个包，一个孩子看到一件玩具……仿佛那一样东西就是能带给我们幸福、快乐、成功的那个源头。亚哈斯在大马士革看见那座坛，就好像看到了已经实现的应许、找到了解决所有问题的答案。于是亚哈斯一回来，就迫不及待地在这座大马士革牌的坛上献祭，把各种祭都献一遍—燔祭、素祭、奠祭、平安祭……然而，亚哈斯献这一切的祭都与敬虔无关，而只与得到他想要的有关。以前的那座坛上，我献祭也得不到我想要的；现在这座坛，我在大马士革亲眼看见了，好使、管用，所以原先那座坛要挪一边去、给这座坛让位。

亚哈斯嘱咐祭司乌利亚：“早晨的燔祭、晚上的素祭、王的燔祭素祭、国内众民的燔祭素祭奠祭，都要烧在大坛上；燔祭牲和平安祭牲的血也要洒在这坛上；只是铜坛我要用以求问耶和华。”(15) 你看，所有的宗教活动还在热火朝天地举行着，从外表上看不出来有太大的异样；但是在那眼所看不见的宝座上，坐着的不再是耶和华，而是亚哈斯自己。所以，亚哈斯那句留着铜坛用来求问耶和华，根本也是假的；他所谓的“求问”，不过是走形式、装样子罢了。这就是为什么耶和华说：“你们所献的许多祭物与我何益呢？公绵羊的燔祭和肥畜的脂油，我已经够了。公牛的血，羊羔的血，公山羊的血，我都不喜悦……可叹忠信的城变为妓女。从前充满了公平、公义居在其中，现今却有凶手居住。你的银子变为渣滓；你的酒用水掺兑。你的官长居心悖逆，与盗贼作伴，各都喜爱贿赂，追求赃私。他们不为孤儿伸冤，寡妇的案件也不得呈到他们面前。”(赛1:11、21-23) 而犹大的全面腐化，也体现在祭司身上发生的变化。乌利雅王的时候，尚有冒死进谏的耶和华勇敢的祭司八十位；到了亚哈斯时，大祭司却随着王一同堕落，还“照着亚哈斯王所吩咐的行”(16)。亚哈斯和犹大的居民，岂不正是“有敬虔的外貌，却背了敬虔的实意”的那种人吗？(提后3:5)

从亚哈斯的身上我们可以看到，信仰是如何渐渐沦为空洞的宗教。一开始，是我们的心偏离神、疏远神；然后我们将那能满足我们实际需要的事物请上宝座；最后，表面上一切宗教活动还照旧，但是心灵里面已经全然被假神所腐蚀，变得自私、冷漠，所行出来的与神的公义、怜悯没有半毛钱关系。亚哈斯表面看起来从亚述那里得着了他想要的，最终却沦为了私欲的奴隶。就如诗篇106篇15节所说，“他将他们所求的赐给他们，却使他们的心灵软弱。”

三、黑暗中的盼望

亚哈斯堕落的最后一步，就是他彻底改造圣殿—他打掉盆座四面镶着的心子，挪移为安息日所盖的廊子。而他之所以这么做，是“因亚述王的缘故”(18)，就是为了避免冒犯亚述王。胆敢随意改造圣殿、将圣殿降格为拜偶像之场所，亚哈斯是第一人、也是最后一人。亚哈斯既然敢挪移、玷污耶和华的圣所，这就离耶和华挪移犹大家不远了。也正是在这样的黑暗之中，神藉着以赛亚应许了那“名称为奇妙、策士、全能的神、永在的父、和平的君，他的政权与平安必加增无穷。他必在大卫的宝座上治理他的国，以公平公义使国坚定稳固，从今直到永远。”(赛9:6-7)

很多年以后，魔鬼在旷野里试探“以马内利”完全应验的那一位，最后一招就是带他上了 一座最高的山，将世上的万国与万国的荣华都指给他看，对他说： ‘你若俯伏拜我，我就把这一切都赐给你。耶稣说：’撒旦’退去吧！因为经上记着说：’当拜主你的神，单要侍奉他。’”(太4:8-10)

所谓敬拜，就是把神所配得的分量归给他。但愿我们在一切的困苦、挣扎当中，坚心仰赖神，因为惟有他才配得我们的心，也惟有他胜过我们心中所想望的一切。
